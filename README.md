# Network Reconnaissance & Analysis Tools

A collection of Python-based network tools developed for network exploration, port scanning, and traffic analysis. This project is built to demonstrate the practical application of socket programming and network protocols in Python.

## ⚠️ Disclaimer
This project is created for educational purposes and authorized security auditing only. The author is not responsible for any misuse or damage caused by these tools. Do not run these scripts against targets without explicit, written permission.

## 🚀 Features
- **Network Scanner:** Discovers active hosts on a local network using ARP requests.
- **Port Scanner:** Efficiently scans target hosts for open ports and identifies potential services.
- **Wi-Fi Analyzer:** Analyzes local wireless networks and available access points.
- **Packet Sniffer:** Captures and inspects raw network packets (requires root/administrator privileges).
